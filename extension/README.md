# extension/

Browser extension (single codebase, Manifest V3) for Chromium (Chrome/Edge/
Brave, desktop) and Firefox (desktop + Android). AO3 real-time capture,
three-state badges, status hooks, and one-time backfill/catch-up commands.

**No code yet** — scaffold stage. First code lands in Phase 3, with status hooks
+ AO3-actions banner in Phase 4 and backfill commands in Phase 7.

Spec: [../docs/components/extension.md](../docs/components/extension.md).
Installed unpacked in developer mode — no store submission.
